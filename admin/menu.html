# 介绍

顾名思义，菜单就是会显示在页面左侧的菜单

菜单可以是一个入口，能够进入指定的某个自定义页面

菜单也可以是一个项目，承载多个页面入口

# 顶级菜单

进入菜单：管理/菜单管理

TIP

通常情况下，一个项目对应着一个顶级菜单

现在，通过右上角的新建菜单按钮来创建一个顶级菜单

点击按钮打开浮层

TIP

不写父级菜单ID就会创建一个顶级菜单

菜单名称不解释了

说一下类型：

  • 类型为菜单的时候，通常表示会有子菜单或子页面

  • 类型为页面的时候，就会访达一个自定义页面了

TIP

菜单类型为页面的时候，需要再填写一个上图中没显示的参数

这里暂时不做说明，放在后面的场景说

# 按钮组

先来创建几个菜单：

可以看到每个菜单后面都有几个按钮，现在从左到右简单介绍一下

  • 复制：复制当前行的菜单id

  • 添加：在当前行菜单（页面没有这个按钮）

  • 编辑：编辑当前行的菜单

  • 删除：删除当前行的菜单
