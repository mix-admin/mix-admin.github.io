# 添加数据库连接

所有的操作都源于数据库，所以首先要从数据库管理开始

进入菜单：管理/数据库管理，点击右上角的按钮添加一个数据库连接

然后会打开一个浮层用来填写相关参数

这里就先添加本地的数据库用于后面的测试(localhost)

基本上所有的参数都不需要介绍。

# 设置别名

每个数据库连接最好都有一个别名

这样在后面你会通过这个名称来快速的找到你想要操作的数据库

别名的设置不是强制性的

如果不设置默认会显示数据库地址

现在还有疑惑没有关系，后面场景会再次提到这点（绑定表）
