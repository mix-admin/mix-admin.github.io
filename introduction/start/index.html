    # Node

    先了解一下数据库的必要条件：

    1、Mysql8.0+

    2、默认连接数据库的配置信息为：

    {
  host: 'localhost',
  user: 'root',
  password: 'mysqladmin',
}

    修改自己数据库的配置（推荐）

    ALTER USER "root"@"localhost" IDENTIFIED WITH mysql_native_password BY "mysqladmin";
FLUSH PRIVILEGES;

    或者去项目的src/config/config.dev文件进行修改

    3、创建数据库mix

    CREATE DATABASE mix;

    如果你的3000端口没有被占用的话，现在可以yarn dev启动nodejs项目了

    # Vue

    前端Vue项目没什么需要注意的条件，安装完依赖以后就可以yarn serve启动项目了
